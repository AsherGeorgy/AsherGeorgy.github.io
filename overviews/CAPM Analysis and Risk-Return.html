Overview

This project provides a brief overview of the Capital Asset Pricing Model (CAPM) and its practical applications in financial analysis. It explores the implementation of CAPM using Walmart Inc, Coca-Cola Co, Lockheed Martin Corp and Pfizer Inc stocks as examples, demonstrating how to calculate beta values and expected returns, as well as assess risk-adjusted returns using the Sharpe Ratio with Python.

Investors require a systematic approach to evaluate the expected returns of securities based on their risk. The CAPM serves as a foundational model in finance for understanding the relationship between risk and return, while the Sharpe Ratio helps investors assess the risk-adjusted performance of their investments. Together, these tools enable more informed investment decisions.

View the full project on Github.

Approach

  1. Data Retrieval: Historical stock prices are fetched using the Yahoo Finance API. Monthly log returns are calculated to understand stock performance over time.
  2. Beta Calculation: The beta values for selected stocks are computed in relation to a market index, indicating the sensitivity of the stock to market movements.
  3. Expected Returns Estimation: Expected returns for each stock are calculated using the CAPM formula, incorporating the risk-free rate and equity risk premium.
  4. Sharpe Ratio Calculation: The Sharpe Ratio is computed to evaluate the risk-adjusted return for the selected stocks, providing insights into their performance relative to the risk taken.
