Overview

The Portfolio Optimization Model is an interactive web application built with Python and Streamlit. It applies Modern Portfolio Theory and Convex Optimization to help users analyze and optimize financial portfolios.

Key Features

  • User-Friendly Interface: Test with randomly generated inputs or run with customized input stock tickers, target growth rates (CAGR), and benchmark indices.
  • Portfolio Analysis:
    • Calculates essential metrics such as returns, volatility, and Sharpe ratios.
    • Simulates and visualizes the Efficient Frontier to highlight optimal portfolios.
  • Portfolio Optimization:
    • Leverages advanced convex optimization techniques to suggest the best portfolio allocation.

  • Visualization Tools:
    • Generate dynamic charts using Plotly to compare portfolio performance.
    • Provides detailed correlation heatmaps and performance breakdowns of individual assets.

Tools and Technologies Used

  • Python Libraries:
    • Streamlit for building interactive web applications.
    • NumPy and Pandas for data manipulation and analysis.
    • Plotly for creating interactive visualizations.
    • cvxpy for convex optimization used in portfolio optimization.
    • Seaborn for statistical data visualization.
  • Yahoo Finance API: Retrieves historical stock and index data for analysis.
  • Modern Portfolio Theory: Calculates risk-adjusted returns using statistical modeling to optimize investment portfolios.
  • Data Visualization:
    • Displays Heatmaps for visualizing asset correlations.
    • Interactive line charts to track cumulative performance over time.
    • Visualizes the Efficient Frontier to identify optimal portfolio allocations.
    • Calculates and displays Sharpe Ratios for portfolio risk-return optimization.

Challenges Solved

  • Streamlit Integration: Integrated the Python script with Streamlit, enabling interactive user input and dynamic visualizations for real-time portfolio analysis and optimization.
  • Dynamic Interface: Developed a user-friendly interface to accommodate input for stock tickers, target growth rates (CAGR), and benchmark indices.
  • Computational Optimization: Implemented Monte Carlo simulations to efficiently process large datasets for robust portfolio analysis.
  • Real-Time Validation: Integrated checks for stock tickers and benchmark indices to ensure the accuracy of the retrieved data.
