Overview

This project implements a momentum-based investment strategy for selecting stocks from the S&P 500 index. It identifies top-performing stocks based on historical returns over multiple time frames and calculates the optimal number of shares to buy for a specified portfolio size. Investing in "momentum stocks"—those with strong recent performance—is a popular strategy among traders and investors. This project addresses the challenge of identifying these top momentum stocks and determining the appropriate share quantities to maximize potential returns by capturing ongoing upward trends.

View the full project on Github.

Approach and Methodology

  1. Data Collection:
    • Fetch the list of S&P 500 companies from Wikipedia.
    • Download one year of daily adjusted closing price data for each S&P 500 stock using the Yahoo Finance API.
  2. Data Analysis:
    • Calculate the one-year return for each stock.
    • Identify the top 50 stocks based on their one-year return.
  3. Momentum Strategy Implementation:
    • Calculate returns over different time frames (one month, three months, six months, and one year).
    • Rank stocks based on these returns and compute percentiles to score momentum.
    • Calculate the HQM (High-Quality Momentum) Score for each stock, which is the average of these percentiles.
  4. Portfolio Construction:
    • Prompt the user for the portfolio size.
    • Calculate the position size for each stock based on equal-weighted investment and the stock's current price.
    • Determine the optimal number of shares to buy for each stock.
  5. Export Results:
    • Output the final list of stocks, their calculated metrics, and the number of shares to buy to an Excel file using xlsxwriter.
