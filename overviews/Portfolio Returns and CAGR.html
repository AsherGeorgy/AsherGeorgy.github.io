Overview

This project provides a comprehensive analysis of individual stock performance and portfolio returns using Python. The analysis includes calculating key metrics such as simple and log returns, cumulative returns, and Compound Annual Growth Rate (CAGR), while also comparing the performance of multiple stocks in a portfolio over a specified period.

Investors need to evaluate both individual stock performance and the overall performance of their investment portfolios to make informed decisions. This project addresses this need by offering a framework to calculate essential metrics for both individual stocks and portfolios.

These metrics help investors understand the risk and return profiles of their investments, enabling them to make better strategic choices.

View the full project on Github.

Approach and Methodology

  1. Data Retrieval:

    Used Yahoo Finance API to fetch historical daily adjusted closing prices for stocks. Conducted analysis over specified periods to calculate returns and growth rates.

  2. Individual Stock Analysis:
    • Simple Returns: Calculated daily and annualized simple returns using percentage change in closing prices.
    • Log Returns: Computed daily and annualized log returns for more accurate risk assessment.
    • Cumulative Return and CAGR: Measured overall return and growth rate of stocks over multiple years.
  3. Portfolio Analysis:
    • Relative Price Movement: Compared relative performance of multiple stocks in a portfolio using normalized price charts.
    • Weighted Average Return: Calculated portfolio return based on asset allocation weights.
    • Cumulative Return and CAGR: Assessed overall portfolio growth and performance over the analysis period.
  4. Visualization:

    Plotted return distributions and price movements to visually compare stock and portfolio performance.
