Overview

This began as a fun personal project during my time playing Clash of Clans which later evolved into a comprehensive tool for players to effectively plan their Hero Equipment upgrades. It helps optimize upgrade paths based on various in-game parameters, such as Townhall Level, Trophy League, blacksmith balance, and Clan War frequency. Upon receiving positive feedback from the Reddit community, I decided to document it on my GitHub.

The tool is a Google Sheets-based calculator that uses formulas and lookup tables to dynamically update based on user input, providing clear visualization of the required resources and time needed for upgrades. The challenge of Clash of Clans is to efficiently plan upgrades due to the complexity of resource management and varying upgrade costs. Balancing resource collection, time, and strategic gameplay decisions is key to quick progress. This project addresses this challenge by offering a structured, user-friendly tool for visualizing and planning upgrades more effectively.

View the full project on Github.

Approach and Methodology

  1. Data Input: Players input their Townhall Level, Trophy League, blacksmith balance, and Clan War frequency.
  2. Dynamic Calculation: The tool uses formulas to dynamically calculate the resources required and the time needed to collect these resources based on user inputs.
  3. Visual Feedback: Provides visual cues and error messages to guide the user in inputting valid data and understanding the output.
  4. Customization: Allows for personalized upgrade paths, enhancing decision-making and strategic planning.
