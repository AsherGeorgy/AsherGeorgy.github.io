Overview

This personal website was developed to showcase my professional resume, portfolio, and contact details in an intuitive and organized manner. Initially conceived as a simple online resume page, the project expanded to a multi-page personal hub, reflecting my self-driven journey and progress in web development.

Development Process

  1. Inspiration & Foundation: The project began with a basic resume template from W3Schools.com. It provided the initial structure and layout for the website.
  2. Learning HTML & CSS: With no formal background in computer science, I took a self-taught approach, learning HTML and CSS through free online resources.
  3. Gradual Expansion: As I gained more confidence, I added new sections, such as the portfolio and contact pages, and enhanced the user experience with better navigation, style adjustments, and embedded widgets.
  4. Customization & Feedback: Generative AI tools played a significant role, providing guidance and suggestions for improvements and helping fine-tune the design.
  5. Technologies Used: The website is built using HTML5, CSS3, and Font Awesome for icons. It was initially hosted locally, and was later pushed to GitHub to be hosted as a GitHub page for wider visibility.

Planned Future Improvements

  • Implementing basic JavaScript for dynamic content and enhanced interactivity.
  • Incorporating analytics to track visitor engagement and website performance.
  • Host it independently using a custom domain.
