Overview

This project involves processing raw bank statements of a firm to create a user-friendly and interactive Excel file that summarizes financial transactions. The Excel file utilizes various data processing and visualization techniques to provide insights into the firmâ€™s financial activities.

Firms often need to analyze their financial transactions to manage cash flow, monitor expenses, and prepare for audits. This project aims to transform raw bank statements into an organized Excel workbook, enhancing readability and enabling interactive exploration of data. This approach allows for quick insights into financial health and supports more efficient decision-making processes.

View the full project on Github.

Approach and Methodology

  1. Data Processing: Imported raw data from text/CSV files using Excel's Power Query Editor and applied data cleaning techniques.
  2. Checking Account Sheet: Created a detailed accounting of monthly transactions with interactive features like slicers.
  3. Individual Totals Sheet: Developed a summary sheet to show totals based on unique entries, using dropdowns and conditional formatting.
  4. Check Register Sheet: Constructed a detailed table for issued checks with slicers for enhanced user interaction.
